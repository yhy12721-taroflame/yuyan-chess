# 🎮 象棋線上對戰 - 獨立版 APP

## ⚡ 快速開始

### 直接打開
在瀏覽器中打開以下文件：
```
public/app-standalone.html
```

或者雙擊文件即可打開。

## 🎯 功能

### 左側面板：房間和玩家
- ✅ 輸入玩家名稱
- ✅ 創建房間
- ✅ 加入房間
- ✅ 查看可用房間列表
- ✅ 查看房間玩家
- ✅ 離開房間

### 右側面板：遊戲
- ✅ 遊戲棋盤
- ✅ 移動歷史
- ✅ 重新開始
- ✅ 悔棋功能

### 標籤功能
- ✅ **遊戲**：棋盤和移動歷史
- ✅ **統計**：遊戲統計信息
- ✅ **消息**：系統消息日誌

## 🎮 使用步驟

### 第 1 步：輸入名稱
在左側面板輸入你的玩家名稱

### 第 2 步：創建或加入房間
- **創建房間**：點擊「創建房間」按鈕
- **加入房間**：輸入房間 ID，點擊「加入房間」

### 第 3 步：開始遊戲
- 房間滿員後自動開始遊戲
- 點擊棋盤格子進行操作
- 查看移動歷史

### 第 4 步：遊戲操作
- **重新開始**：重置遊戲
- **悔棋**：撤銷上一步移動
- **離開房間**：退出當前房間

## 📊 功能說明

### 房間管理
- 自動生成房間 ID
- 支持多個房間並發
- 房間滿員後自動開始遊戲

### 玩家管理
- 顯示房間中的所有玩家
- 實時更新玩家列表
- 支持玩家加入和離開

### 遊戲功能
- 簡化的棋盤顯示
- 移動歷史記錄
- 遊戲時長計時
- 棋子計數

### 消息系統
- 實時系統消息
- 彩色編碼（成功、錯誤、警告、信息）
- 消息清空功能

## 💡 使用建議

### 單人測試
1. 打開應用
2. 輸入名稱
3. 創建房間
4. 查看房間信息

### 多人測試
1. 打開多個瀏覽器標籤
2. 每個標籤輸入不同的名稱
3. 第一個標籤創建房間
4. 其他標籤加入房間
5. 房間滿員後開始遊戲

## 🎯 測試場景

### 場景 1：創建房間（1 分鐘）
```
1. 輸入玩家名稱
2. 點擊「創建房間」
3. 查看房間 ID
4. 驗證房間已創建
```

### 場景 2：加入房間（2 分鐘）
```
1. 打開第二個標籤
2. 輸入不同的玩家名稱
3. 輸入房間 ID
4. 點擊「加入房間」
5. 驗證兩個玩家在同一房間
```

### 場景 3：遊戲操作（1 分鐘）
```
1. 點擊棋盤格子
2. 查看移動歷史
3. 點擊「悔棋」
4. 點擊「重新開始」
```

### 場景 4：多房間並發（2 分鐘）
```
1. 打開多個標籤
2. 每個標籤創建不同的房間
3. 驗證房間之間的隔離
4. 驗證玩家列表正確
```

## 🔧 故障排除

### 無法打開文件
- 確保文件路徑正確：`public/app-standalone.html`
- 使用現代瀏覽器（Chrome、Firefox、Safari、Edge）
- 檢查文件是否存在

### 房間無法創建
- 確保輸入了玩家名稱
- 檢查瀏覽器控制台是否有錯誤
- 刷新頁面重試

### 無法加入房間
- 確保房間 ID 正確
- 確保房間未滿員
- 確保輸入了玩家名稱

### 棋盤無法顯示
- 刷新頁面
- 清除瀏覽器緩存
- 使用不同的瀏覽器

## 💻 系統要求

- 現代瀏覽器（Chrome、Firefox、Safari、Edge）
- JavaScript 已啟用
- 無需服務器或網絡連接
