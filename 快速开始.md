## 需要做的事情

### 1. 验证 Node.js 安装

**方法 1：运行检查脚本**
双击运行 `check-setup.bat` 文件

**方法 2：手动检查**
打开**新的** PowerShell 或命令提示符窗口，运行：
```bash
node --version
npm --version
```

如果显示版本号（如 v20.x.x），说明安装成功！

### 2. 如果命令找不到

这是因为环境变量还没有生效。请尝试：

1. **关闭当前的 PowerShell/命令提示符**
2. **重新打开一个新的窗口**
3. 再次运行 `node --version`

如果还是不行：
- **重启电脑**（这会确保环境变量生效）

### 3. 安装项目依赖

Node.js 可用后，在项目目录运行：

```bash
npm install
```

这会安装所有需要的包（可能需要几分钟）。

### 4. 运行测试

```bash
npm test
```

你应该看到所有测试通过：
- ✅ Position 单元测试（30+ 个测试）
- ✅ Position 属性测试（Property 1, 2, 3）

### 5. 查看测试覆盖率

```bash
npm test -- --coverage
```

## 遇到问题？

### npm 命令找不到
- 确保 Node.js 已安装
- 重启终端或电脑
- 检查环境变量中是否有 Node.js 路径

### 测试失败
- 确保运行了 `npm install`
- 检查是否有错误信息

### 其他问题
查看 `SETUP.md` 获取详细的故障排除指南。
