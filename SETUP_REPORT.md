## 🔧 故障排除

### 常見問題

| 問題 | 解決方案 |
|------|---------|
| 無法連接 | 確保服務器已啟動：`npm run server` |
| 無法創建房間 | 確保已連接到服務器 |
| 無法加入房間 | 確保房間 ID 正確 |
| 消息無法發送 | 確保已連接到服務器 |
| 其他問題 | 查看日誌和瀏覽器控制台 |

### 調試技巧

1. **查看瀏覽器控制台**：按 F12 打開開發者工具
2. **查看服務器日誌**：在終端中查看服務器輸出
3. **監控日誌**：使用測試頁面的日誌功能
4. **查看統計信息**：實時監控消息流
5. **檢查最後接收的消息**：驗證數據格式
