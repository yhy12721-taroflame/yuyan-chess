## 🎯 快速開始

### 第 1 步：打開應用
```
public/app-standalone.html
```

### 第 2 步：輸入名稱
在左側面板輸入你的玩家名稱

### 第 3 步：創建或加入房間
- **創建房間**：點擊「創建房間」
- **加入房間**：輸入房間 ID，點擊「加入房間」

### 第 4 步：開始遊戲
- 房間滿員後自動開始
- 點擊棋盤進行操作
- 查看移動歷史

```

4. 驗證玩家列表
```
